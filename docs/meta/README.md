# Meta Documentation

This directory contains **documentation about the documentation** - meta-level records about how the docs are organized, standards, and reorganization efforts.

## Contents

- `documentation-reorganization-2025-12-21.md` - Complete record of the Dec 21, 2025 documentation reorganization (testing/ and sprint-artifacts/ restructuring)

## Purpose

This folder holds:
- **Organization records** - When and why directories were restructured
- **Documentation standards** - Conventions and naming patterns
- **Refactoring notes** - Historical context for future maintainers
- **Meta-guides** - How to organize and maintain the docs

## Why This Exists

As the project grows, the documentation structure evolves. This directory preserves the **history and rationale** behind organizational decisions, making it easier for:
- **Future team members** to understand why things are organized this way
- **AI agents** to learn documentation conventions
- **Maintainers** to know when it's time for another reorganization
