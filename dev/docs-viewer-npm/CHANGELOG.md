# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.3] - 2025-01-XX

### Changed
- Updated repository URLs to point to `weave-prod` repository
- Configured package for GitHub Packages registry
- Added `.npmrc` configuration for GitHub Packages authentication

### Added
- Added `publishConfig` to `package.json` for GitHub Packages registry
- Added `CHANGELOG.md` file

## [0.1.2] - Previous Version

### Features
- GitHub-style Markdown rendering
- Dark mode with smooth transitions
- Syntax highlighting (Prism.js)
- Folder tree navigation
- Zero-config CLI usage (`docs-viewer` / `npx`)
- Cross-platform support (Windows, macOS, Linux)
- Node.js and Python server implementations
- Path traversal protection for file access
